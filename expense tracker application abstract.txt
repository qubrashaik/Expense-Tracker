Managing personal finances has become increasingly challenging due to the growing number of daily financial transactions and the need for effective budgeting. Many individuals find it difficult to record and monitor their income and expenses regularly, often relying on traditional methods such as notebooks or spreadsheets. These methods are time-consuming, prone to errors, difficult to maintain, and lack analytical capabilities that help users understand their spending patterns. To address these challenges, this project presents the development of an "Expense Tracker Application", a secure and efficient backend system designed to simplify personal financial management. The application enables users to record, organize, update, and analyze their financial transactions through a centralized platform while ensuring data security and easy accessibility. The primary problem addressed by this project is the lack of a reliable and automated system that allows users to efficiently manage expenses, income, and budgets while providing meaningful financial insights. Existing systems often offer limited functionality, weak security mechanisms, or inadequate reporting features, making it difficult for users to make informed financial decisions. The developed application overcomes these limitations by providing secure RESTful APIs that support efficient transaction management and financial reporting. The project is implemented using "Java Spring Boot" as the backend framework, Spring Security for authentication and authorization, JWT (JSON Web Tokens) for secure session management, "Spring Data JPA (Hibernate)" for database operations, MySQL as the relational database management system, Maven for dependency management, and Postman for API testing and validation. The application follows a layered architecture consisting of the Controller, Service, Repository, and Database layers, where the Controller handles client requests, the Service layer contains business logic, the Repository layer interacts with the database, and MySQL stores all application data securely. The workflow begins with user registration and authentication, after which a JWT token is generated to provide secure access to protected API endpoints. Authenticated users can create, update, retrieve, and delete expense and income records, categorize transactions, manage budgets, and filter financial data based on categories and date ranges. The system also generates monthly summaries and category-wise expenditure reports that help users monitor spending habits and improve financial planning. The developed Expense Tracker Application provides a secure, scalable, and user-friendly backend solution for personal finance management while demonstrating the practical implementation of RESTful API development, secure authentication, database integration, and layered software architecture using Spring Boot and MySQL. The application serves as a reliable foundation for future integration with web or mobile frontend applications and can be further enhanced with features such as graphical dashboards, notifications, cloud deployment, and advanced financial analytics.
